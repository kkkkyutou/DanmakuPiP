# 安全审查记录（V1）

## 审查日期
- 2026-03-31

## 审查范围
- `manifest.json` 权限与 host 范围
- 运行时代码（background/content/popup/options）
- 发布脚本与打包产物

## 审查结论
- 当前版本满足“轻量、本地处理、最小权限”原则，可作为 V1 发布候选。

## 检查项与结果
1. 权限最小化
- 仅声明 `storage`、`activeTab`。
- 未声明 `tabs`、`webRequest`、`downloads` 等高敏权限。
- 结果：通过。

2. 远程代码执行风险
- 未使用 `eval`、`new Function`、远程脚本注入。
- 无动态下载后执行逻辑。
- 结果：通过。

3. 隐私数据采集风险
- 未采集账号密码、表单输入、剪贴板、定位等敏感信息。
- 仅处理视频时间与公开弹幕/时间文本。
- 结果：通过。

4. 网络访问边界
- 仅访问 B站与 YouTube 相关站点。
- 未接入第三方统计/上报服务。
- 结果：通过。

5. 本地存储安全
- 仅存储功能设置项（字号、速度、屏蔽词等）。
- 不存储身份令牌或敏感凭据。
- 结果：通过。

## 已知剩余风险
- YouTube 页面结构变化可能影响时间文本解析。
- 站点接口变更可能导致弹幕抓取失败，需要版本跟进。

## 发布建议
- 可发布 V1，并在商店说明中明确已知限制与支持范围。
