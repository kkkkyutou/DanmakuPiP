# 隐私说明（V1）

## 数据处理原则
- 本扩展默认本地运行，不上传弹幕正文与观看记录到开发者服务器。
- 扩展不采集账号密码、支付信息、浏览器历史记录。
- 设置项仅保存在浏览器本地 `chrome.storage.local`。

## 使用到的数据
- 页面视频播放状态（播放/暂停/进度/倍速）。
- 站点公开弹幕或时间文本数据。
- 用户自行填写的本地配置（字号、速度、屏蔽词等）。

## 权限说明
- `activeTab`：用于在当前标签页发送启动/停止命令。
- `storage`：用于保存本地设置。
- Host 权限：
  - `*.bilibili.com`、`api.bilibili.com`
  - `*.youtube.com`
  - 仅用于读取视频相关公开数据并渲染弹幕。

## 不做的事情
- 不进行用户身份追踪。
- 不将数据出售或共享给第三方。
- 不注入远程脚本，不执行动态下载执行代码。
